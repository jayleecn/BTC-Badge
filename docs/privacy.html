Privacy Policy for BTC Badge

Last updated: December 12, 2024

1. Information Collection and Use

BTC Badge is designed with privacy in mind and operates with minimal data collection. The extension only retrieves publicly available Bitcoin price data from bitcoinition.com to display the current Bitcoin price in your browser's toolbar.

2. Data Storage

We use Chrome's storage API solely for:

  • Storing your display preferences
  • Temporarily caching Bitcoin price data to minimize API calls

All data is stored locally on your device and is never transmitted to external servers.

3. Permissions

BTC Badge requires minimal permissions to function:

  • Internet access: To fetch Bitcoin price data
  • Storage: To save your preferences locally

4. Third-Party Services

We only interact with bitcoinition.com to fetch Bitcoin price data. No personal information is shared with this or any other third-party service.

5. Updates to Privacy Policy

We may update this privacy policy as needed. Significant changes will be communicated through the Chrome Web Store and this page.

6. Open Source

BTC Badge is open source. You can review our code on GitHub to verify our privacy practices.

7. Contact

If you have any questions about this privacy policy or the BTC Badge extension, please open an issue on our GitHub repository.
